## **Nome do projeto**
Chatbot para Gestão Inteligente de Eletropostos Comerciais

## **Problema abordado**
Resolver a falta de integração e automação na gestão de eletropostos, especialmente em relação ao controle de carregamento, distribuição de energia, registro de uso e suporte aos diferentes usuários envolvidos.

## **Proposta do chatbot**
Atuar como assistente especializado para orientar usuários sobre implantação, operação e gestão de eletropostos, adaptando as respostas conforme o perfil de quem consulta (operador, síndico, morador ou técnico).

## **Tecnologias selecionadas**

* Python
* `huggingface_hub`
* `InferenceClient`
* `ipywidgets`
* `IPython.display`
* Kaggle Secrets
* Modelo de linguagem `openai/gpt-oss-120b`
* Notebook Jupyter/Kaggle

## **Justificativa técnica**

* **Python:** prototipação rápida e amplo suporte para IA
* **Hugging Face + InferenceClient:** acesso simples a modelos avançados sem infraestrutura própria
* **ipywidgets + IPython.display:** criação de interface interativa dentro do notebook
* **Kaggle Secrets:** armazenamento seguro da chave de API
* **Modelo grande de linguagem:** maior capacidade de contextualização e adaptação de respostas
* **Notebook Kaggle/Jupyter:** ambiente ideal para testes, demonstrações e desenvolvimento acadêmico rápido
